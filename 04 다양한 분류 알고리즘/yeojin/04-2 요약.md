# 04-2 확률적 경사 하강법(Stochastic Gradient Descent)

### Gradient Descent Algorithm
$W := W -\alpha\frac{\partial}{\partial W} cost(W)$
<br>$\rightarrow$ 최적의 $W$를 찾는 것이 목적
- Stochastic Gradient Descent: 샘플 하나하나씩 업데이트
- Minibatch Gradient Descent: 여러개의 샘플(minibatch)을 사용해 업데이트
- Batch Gradient Descent: 샘플 전체를 사용해 한 번에 업데이트, 연산량 많음

### 손실 함수(loss function)
- cost 함수라고 하기도 함
- 종류: 로지스틱 손실 함수(이진 분류), 크로스엔트로피 손실 함수(다중 분류), mean squared error(회귀) 등등

### 에포크(epoch)
- 훈련 세트를 한 번 모두 사용하는 과정
- 에포크 횟수가 적으면 과소적합 위험
- 에포크 횟수가 많으면 과대적합 위험
- 과대적합으로 가기 전에 훈련을 멈추는 것을 조기종료(early stopping)이라고 함

## 문법
### 확률적 경사 하강법
```python
from sklearn.linear_model import SGDClassifier

sc = SGDClassifier(loss='log_loss', max_iter=10, random_state=42) # max_iter: 에포크 횟수
sc.fit(train_input, train_target)
```
### partial_fit()
- 이어서 학습하는 법
```python
sc.partial_fit(train_input, train_target)
```
