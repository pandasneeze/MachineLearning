# 04-1 로지스틱 회귀
- 클래스 예측을 0과 1이 아닌 확률값으로 계산하는 것
- sigmoid $\phi = \frac{1}{1+e^{-z}}$
- softmax $y_k= \frac{e^{z_k}}{\sum_j{e^{z_j}}}$
- $Z = WX + b$ <br>

| | 의미 | shape |
|---|---|---|
|$Z$|각 클래스별 확률|(클래스 개수, 샘플 개수)|
|$W$|weight|(클래스 개수, feature 개수)|
|$X$|training data|(feature 개수, 샘플 개수)|
|$b$|bias|(샘플 개수, )|

- 이진 분류에서는 시그모이드함수, 다중 분류에서는 소프트맥스 함수에 z값을 통과시켜 확률 계산
## 문법

### 로지스틱 회귀
```python
from sklearn.linear_model import LogisticRegression
lr = LogisticRegression()
lr.fit(train_input, train_target)
```
### 예측 확률 출력
```python
proba = lr.predict_proba(test_input[:5])
```
