# 03-2 선형 회귀
- **선형 회귀(Linear Regression)**: 훈련 세트에 잘 맞는 직선의 방정식을 찾는 것
- **다항 회귀(Polynomial Regression)**: 훈련 세트에 잘 맞는 다항식을 찾는 것
$Y=XW+b, \ Y: \text{output, target}, X: \text{input, feature}, W:\text{weight}, b: \text{bias}$

## 문법
### 사이킷런
- 선형 회귀 클래스 임포트
```python
from sklearn.linear_model import LinearRegression
```
- `.coef_`, `.intercept_`: $Y = XW + b$ 식에서 $W$, $b$
