# 03-2 선형 회귀
- **선형 회귀(Linear Regression)**: 훈련 세트에 잘 맞는 직선의 방정식을 찾는 것
- **다항 회귀(Polynomial Regression)**: 훈련 세트에 잘 맞는 다항식을 찾는 것
### $\ \ \ Y=XW+b, \ (Y: \text{target}, X: \text{feature}, W:\text{weight}, b: \text{bias})$
<mark>위 식에서 최적의 $W$, $b$를 찾는 것이 선형회귀의 목표임</mark>
## 문법
### 사이킷런
- 선형 회귀 클래스 임포트
```python
from sklearn.linear_model import LinearRegression
```
- `.coef_`, `.intercept_`: $Y = XW + b$ 식에서 $W$, $b$
