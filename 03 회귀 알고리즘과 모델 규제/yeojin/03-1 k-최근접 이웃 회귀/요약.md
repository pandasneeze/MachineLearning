# 03-1 k-최근접 이웃 회귀
- **회귀(regression)**: 클래스 중 하나로 분류하는 것이 아니라 임의의 어떤 숫자를 예측하는 것
| |k-최근접 이웃 알고리즘|k-최근접 이웃 회귀|
|---|---|---|
|feature|생선의 길이, 무게|농어의 길이|
|target|생선의 이름|농어의 무게|
|score() 출력|정확도|결정계수|
- **결정계수($R^2$)**: 예측이 타깃에 가까울 수록 1과 가까운 값을 가짐
- **과대적합(overfitting)**: 훈련 세트에서 점수가 높았으나 테스트 세트에서 점수가 굉장히 나쁜 경우 -> 이웃의 개수를 늘림(일반적인 패턴을 따르게 됨)
- **과소적합(underfitting)**: 훈련 세트보다 테스트 세트의 점수가 높거나 두 점수가 너무 낮은 경우  -> 이웃의 개수를 줄임(국지적 패턴에 민감)

## 문법
### 사이킷런
- `mean_absolute_error()`: 절대값 오차의 평균 반환
```python
mae = mean_absolute_error(test_target - test_prediction)
```
### 넘파이
- `reshape()`: 배열의 크기를 바꾸어줌, -1을 지정하면 자동으로 차원에 맞게 지정
