# 03-3 특성 공학과 규제
- 특성을 많이 추가하면 더 좋은 성능의 머신을 만들 수 있음(`PolynomialFeatures()`클래스의 `transform()`)
- 그러나 overfitting 위험 -> 규제 도입
- **릿지 회귀**와 **라쏘 회귀**로 규제할 수 있음
- 릿지는 계수의 제곱한 값을 기준으로 규제
- 라쏘는 계수의 절대값을 기준으로 규제, 계수를 0으로 만들 수 있음

## 문법
### 판다스
- `read_csv('주소')`: csv 파일을 내려받아 데이터프레임에 저장(`to_numpy()`로 넘파이 배열로 바꿀 수 있음)
- `.head()`: 처음 다섯개의 행 출력

### 사이킷런
- `transform(리스트)`: 새로운 특정을 만들어 줌
```python
from sklearn.preprocessing import PolynomialFeatures

poly = PolynomialFeatures()
poly.fit(train_input)
train_poly = poly.transform(train_input)
test_poly = poly.transform(test_input)
```
- `get_feature_names_out()`: 특성이 각각 어떤 입력으로 만들어졌는지 알려줌
#### 정규화
```python
from sklearn.preprocessing import StandardScaler
ss = StandardScaler()
ss.fit(train_input)
train_scaled = ss.transform(train_input)
test_scaled = ss.transform(test_input)
```
#### 릿지 회귀
```python
from sklearn.linear_model import Ridge
ridge = Ridge()
ridge.fit(train_input, train_target)
```

#### 라쏘 회귀
```python
from sklearn.linear_model import Lasso
lasso = Lasso()
lasso.fit(train_input, train_target)
```
