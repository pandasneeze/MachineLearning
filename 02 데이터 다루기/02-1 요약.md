# 02-1 훈련 세트와 테스트 세트

## 용어
- **지도 학습(supervised learning)**: 정답(target)이 있는 학습법
- **비지도 학습(unsupervised learning)**: 정답이 없는 학습법
- 입력으로 features들을 넣었을 때 정답(target 또는 label)을 output으로 뱉어냄
- 훈련에 사용되는 데이터를 훈련 데이터, 테스트에 사용되는 데이터를 테스트 데이터라고 함
- **샘플(sample)**: 하나의 생선 데이터(하나의 생선 데이터에는 생선의 길이와 무게가 담겨있음)

## 샘플링 편향
- training data set에 샘플이 골고루 섞여있지 않으면 컴퓨터가 제대로 학습하지 못함
- 따라서 서로 다른 타겟의 샘플들을 적절히 섞어주어야 함

## 문법
### 넘파이
- `array()`: 리스트를 넣으면 **넘파이 배열**로 바꾸어줌
```python
# 예시
arr = np.array(arr)
```
- `.shape`: shape을 알려줌
```python
# 예시
print(arr.shape)
```
- `arange()`: 정수 N을 전달하면 0부터 N-1까지 증가하는 배열을 만듦(시작, 종료, 증가까지 정할 수 있음)
- `shuffle()`: random 패키지 안에 있음, 주어진 배열을 무작위로 섞음
```python
# 예시
np.random.seed(42)
index = np.arrange(49)
np.random.shuffle(index)
```
- **배열 인덱싱**: 인덱스 자리에 리스트를 넣으면 여러개의 행(sample)을 선택할 수 있음
```python
# 예시
print(input_arr[[1, 3]]) # 두 번째와 네 번째 샘플 선택

# ------ 출력 ------
[[ 26.3  290. ],
 [ 29.   363. ]]
```
