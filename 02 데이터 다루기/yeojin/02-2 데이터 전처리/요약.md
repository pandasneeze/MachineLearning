# 02-2 데이터 전처리
- 각 특성의 데이터 스케일이 다르면 제대로 된 결과가 나오지 않음
- 특성들 간의 데이터 스케일을 맞춰주는 작업을 **데이터 전처리**라고 함
- **표준점수**: 각 특성값이 평균에서 표준편차의 몇 배만큼 떨어져 있는지를 나타냄
    - $표준점수 = (input - mean) / std$
- **브로드캐스팅**: 크기가 다른 넘파이 배열에서 자동으로 사칙 연산을 모든 행이나 열로 확장하여 수행하는 기능

## 문법
### 넘파이
- `column_stack()`: 전달 받은 리스트를 일렬로 세운 다음 나란히 연결, **튜플로 전달**
```python
np.column_stack([1, 2], [3, 4])

# 결과
[[1, 3],
 [2, 4]]
```
- `ones()`, `zeros()`: 원하는 길이의 1과 0 배열을 만들어줌
- `concatenate()`: 두 배열을 이어붙여줌, **튜플로 전달**
- `mean()`: 평균 계산, axis 지정 가능
- `std()`: 표준편차 계산, axis 지정 가능
```python
mean = np.mean(train_input, axis=0)
std = np.std(train_input, axis=0)
```
### 사이킷런
- `train_test_split()`: 리스트나 배열을 훈련 세트와 테스트 세트로 나누어 줌
    - `random_state` 매개변수를 지정하면 랜덤 시드를 정할 수 있음
    - 기본적으로 25%를 테스트 데이터로 떼어냄
    - `stratify` 매개변수에 타깃 데이터를 전달하면 클래스 비율에 맞게 데이터를 나눔
```python
train_input, test_input, train_target, test_target = train_test_split(
    fish_data, fish_target, stratify=fish_target, random_state=42)
```

### matplotlib
- `xlim()`, `ylim()`: x축, y축 스케일을 바꾸어줌, 튜플로 범위 전달(예: `plt.xlim((0, 1000))`)
