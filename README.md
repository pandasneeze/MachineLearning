# 머신러닝 스터디
혼자 공부하는 머신러닝 + 딥러닝을 함께 공부하는 스터디입니다.

## Study Flow
1. 매주 정해진 분량 학습
2. 개인별 코드/노트 정리
3. Pull Request 제출
4. 주간 미팅에서 질의응답

## Rules
### 1. 레포 클론
```
git clone https://github.com/pandasneeze/MachineLearning.git
```
### 2. 자기 이름 브랜치 만들기(최초 1회)
```
git branch 자기 이름
git push -u origin 자기 이름
```
⚠️ master로 직접 push 금지

### 3. 자신의 브랜치, 폴더에서만 작업
- `git branch`: 현재 브랜치 확인 가능
- 파일 구조 예
```
03 회귀 알고리즘/
 └─ 이름/
    ├─ 내용 정리.md
    └─ linear_regression.ipynb
```
### 4. PR 올리기(미팅 전까지)
**PR 내용**
- 무엇을 배웠는지 간단하게 요약
- 이해 안 된 점
- 토론하고 싶은 점
