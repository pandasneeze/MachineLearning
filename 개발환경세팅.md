#### ⚠️ *& : 이 시스템에서 스크립트를 실행할 수 없으므로...* 와 같은 오류가 나는 경우
- 시작 -> powershell 마우스 오른쪽 버튼 -> 관리자 권한으로 실행 -> 아래 명령어 입력
```
Set-ExecutionPolicy RemoteSigned
```
- `y` 입력 후 엔터
- vs code 껐다 다시 실행
- 터미널 창의 powershell 옆 + 아이콘 클릭, (ml_env) 확인

## 4. 라이브러리 설치
- 터미널에 아래 명령어 붙여넣기
```
pip install numpy matplotlib scikit-learn
pip install tensorflow keras
pip install torch torchvision torchaudio
pip install transformers
```
  
## 주의
- vs code 실행마다 터미널에 (ml-env) 뜨는지 확인
- 안 뜨는 경우 터미널 창의 powershell 옆 + 아이콘 클릭
