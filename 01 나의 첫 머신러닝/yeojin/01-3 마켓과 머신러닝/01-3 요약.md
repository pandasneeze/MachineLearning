# 01-3  마켓과 머신러닝
- 문제: 생선의 크기와 무게가 주어졌을 때, 어떻게 그 생선이 도미인지 빙어인지 알 수 있을까?
- 해결: **k-최근접 이웃 알고리즘**을 이용
- class(lable): 도미, 빙어
- feature: 무게, 길이

## 해결 과정
1. 도미 데이터 준비하기
- 도미 35마리의 길이, 무게 데이터셋을 준비
2. 빙어 데이터 준비하기
- 빙어 14마리의 길이, 무게 데이터셋 준비
3. 두 데이터를 합치기
- 해당 데이터는 컴퓨터를 훈련시키는 trainning data set이 됨
- 행은 각 생선 개체(sample data)를 의미하고 열은 길이, 무게가 됨 -> 49 x 2 matrix
4. 정답 데이터 준비하기
- 위에서 준비했던 각 sample이 도미인지, 빙어인지 컴퓨터에게 알려주어야 함
- 도미는 1, 빙어는 0
- 생선이 49마리이므로 정답 데이터는 49 x 1 matrix
5. 임의의 생선 데이터 입력해보고 컴퓨터가 잘 구분하는지 확인

## k-최근접 이웃 알고리즘
- 주위의 데이터로 현재 데이터를 판단
- 새로운 데이터에 대해 예측할 때는 가장 가까운 직선거리에 어떤 데이터가 있는지 판단
- 데이터가 아주 많은 경우 사용하기 어려움
- 훈련할 때 정답(lable)을 주고 훈련하므로 지도 학습(supervised learning)에 속함

## 문법
### 사이킷런
- `KNeighborsClassifier()`: k-최근접 이웃 알고리즘 클래스
- `fit()`: 훈련 메서드
* input: data set, target
- `score()`: 모델 평가 메서드, 0에서 1사이의 값 반환, 숫자가 클 수록 많이 맞췄다는 의미
* input: data set, target
- `predict()`: 새로운 데이터의 정답 예측
* input: test data
### matplotlib
- `scatter()`: 산점도를 그리는 함수, x축 값과 y축 값을 매개변수로 전달
- `xlabel()`: x축 이름 지정
- `ylabel()`: y축 이름 지정
- `show()`: 그래프 보여주기
